# Как просмотреть чеки

В личном кабинете Prodamus можно отследить все чеки по транзакциям, узнать их статус и найти нужный чек с помощью фильтров. В этой инструкции рассказываем, как это сделать.

**Содержание:**

1. [Просмотр всех чеков](kak-prosmotret-cheki.md#prosmotr-vsekh-chekov)
2. [Подробная информация по чеку](kak-prosmotret-cheki.md#podrobnaya-informaciya-po-cheku)
3. [Фильтрация и поиск чеков](kak-prosmotret-cheki.md#filtraciya-i-poisk-chekov)

### Просмотр всех чеков

В меню слева выберите раздел «Чеки».

<figure><img src="../.gitbook/assets/unknown (171).png" alt=""><figcaption></figcaption></figure>

Откроется таблица со всеми чеками. Она делится на несколько колонок:

* Ссылка на чек — активная ссылка, ведущая на страницу чека. Нажав на иконку копирования справа, можно скопировать URL.
* Сумма — сумма чека в рублях.
* Дата и время — когда был сформирован чек.
* Номер транзакции — уникальный идентификатор, который соответствует транзакции, по которой создан чек.
* Тип — тип операции:

&#x20;           — Приход — чек по поступлению оплаты.

&#x20;           — Возврат — чек по возврату средств.

* Статус — текущее состояние чека:

&#x20;           — Сформирован — чек успешно создан.

&#x20;           — В обработке — чек ещё не готов.

&#x20;           — Ошибка — при формировании чека произошла ошибка.

<figure><img src="../.gitbook/assets/unknown (172).png" alt=""><figcaption></figcaption></figure>

### Фильтрация и поиск чеков

Если чеков много, для поиска используйте панель фильтров над таблицей. Чеки можно отфильтровать по следующим параметрам:

* Номер заказа, почта, номер телефона или продукт&#x20;
* Канал продаж — сайт, магазин или другая точка продаж, через которую прошла транзакция.
* Дата создания — конкретная дата или диапазон.
* Продукт — товар или услуга, за которую сформирован чек.

<figure><img src="../.gitbook/assets/unknown (174).png" alt=""><figcaption></figcaption></figure>

**Чтобы найти чек** воспользуйтесь поиском, либо нажмите на нужный фильтр, выберите параметры, по которому нужно применить сортировку и нажмите «Применить».

<figure><img src="../.gitbook/assets/unknown (175).png" alt=""><figcaption></figcaption></figure>

Чтобы вернуть полный список чеков, используйте кнопку «Сбросить фильтры».
