# Описание разделов личного кабинета Prodamus.Kassa

[https://new.prodamuskassa.runew.prodamuskassa.ru](https://new.prodamuskassa.ru/)

#### 1. Раздел – Мои компании

В данном разделе вы можете:

• Получить «Настройки интегратора»;

• Просмотреть данные вашей компании;

• Узнать, сколько касс закреплено за вашей компанией, и их текущий статус.

![](<../.gitbook/assets/image (82)>)

#### 2. Раздел – Кассы

Этот раздел содержит информацию о вашей кассе:

• Название компании в ЛК;

• Ссылка на интернет-магазин;

• ЗН ККТ (заводской номер контрольно-кассовой техники);

• РНМ (регистрационный номер модели);

• Модель ККТ;

• ЗН ФН (заводской номер фискального накопителя);

• Номер устройства;

• Статус кассы;

• Код активации ОФД;

• Дата фискализации;

• Адрес места установки ККТ.

![](<../.gitbook/assets/image (83)>)

#### 3. Раздел – Формирование чеков (1.05/1.2)

В этом разделе вы можете вручную сформировать чек, заполнив необходимые поля.

![](<../.gitbook/assets/image (84)>)

Если касса еще не зарегистрирована, при открытии данного раздела может появиться соответствующее уведомление.

![](<../.gitbook/assets/image (85)>)

Если у вас есть магазины с фискализированными под версию ФФД 1.05 или 1.2 кассами, то после перехода откроется страница «Общие данные чека» с формой для создания чека. Обязательные для заполнения поля помечены символом «\*».

**Общие данные чека**

После перехода на страницу «Общие данные чека» вам предлагается выбрать:

− компанию и магазин, для которого необходимо сформировать чек,

− систему налогообложения, которая будет отображаться в чеке,

− тип чека.

Также необходимо указать ФИО кассира, который формирует чек.

![](<../.gitbook/assets/image (86)>)

**Выбор компании и магазина**

Компания. Выберите в выпадающем списке «Компания» одну из ваших компаний, созданных в ЛК. После выбора компании станет доступен выбор магазина и системы налогообложения.

![](<../.gitbook/assets/image (87)>)

Магазин. Список не доступен для выбора, пока не выбрана компания. Выберите один из магазинов, закрепленных за указанной компанией. Список будет содержать все адреса магазинов, указанные через ";" при добавлении интернет-магазина.

![](<../.gitbook/assets/image (88)>)

Система налогообложения – выпадающий список. Список не доступен для выбора, пока не выбрана компания. Выберите систему налогообложения из доступных для указанной компании.

![](<../.gitbook/assets/image (89)>)

**Добавление кассира**

Ваше ФИО. Добавьте ФИО кассира, который заполняет чек.

![](<../.gitbook/assets/image (90)>)

**Выбор типа чека**

Тип чека. Выберите в списке тип чека, который необходимо сформировать:

− Приход,

− Возврат прихода,

− Коррекция прихода,

− Расход,

− Возврат расхода,

− Коррекция расхода

![](<../.gitbook/assets/image (91)>)

После выбора чека нужного типа отобразятся все необходимые поля для формирования конкретного документа.

#### 4.Раздел – Журнал чеков

Здесь вы можете отслеживать чеки, используя фильтры:

• Статус чека;

• Дата регистрации чека на кассе;

• Номер ФН;

• Тип чека и другие параметры.

![](<../.gitbook/assets/image (92)>)

#### 5. Раздел – Загрузка чеков из файлов XML

Если вы знакомы с основами структуры и синтаксиса XML-файлов, вы можете использовать этот раздел для загрузки чеков. Для создания XML-файла вручную или с помощью программного кода, требуются соответствующие технические навыки.

![](<../.gitbook/assets/image (93)>)

#### 6. Раздел – Мониторинг касс

Здесь представлена подробная информация о кассах и фискальных накопителях (ФН).

![](<../.gitbook/assets/image (94)>)

Для проверки работоспособности кассы нажмите на окно с РНМ, которое раскроет детальную информацию о кассе.

![](<../.gitbook/assets/image (95)>)

• Обновление ФФД;
