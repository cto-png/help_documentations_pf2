# Как подключить платёжный виджет Prodamus

Платёжный виджет Prodamus — это способ принимать оплату прямо на вашем сайте, без перенаправления клиента на отдельную платежную страницу. Виджет открывается во всплывающем окне и полностью берёт на себя процесс оплаты.

Использование платёжного виджета позволяет:

* принимать оплату без редиректов
* сохранить клиента на сайте до момента завершения платежа
* повысить конверсию за счёт более короткого пути до оплаты

### Что нужно для подключения платёжного виджета

Перед началом подключения убедитесь, что у вас есть:

* Активный аккаунт в Prodamus.
* Доступ к HTML-коду страницы сайта.
* Идентификаторы:

1. **merchantId** — идентификатор мерчанта;
2. **salesChannelId** — идентификатор канала продаж.

#### Где получить merchantId и salesChannelId

Идентификаторы **merchantId** и **salesChannelId** выдаются технической поддержкой Prodamus.Чтобы получить их, обратитесь в поддержку любым удобным способом:

* Через бота в MAX: [https://max.ru/id1215156909\_2\_bot](https://max.ru/id1215156909_2_bot)
* По email: [sales@prodamus.ru](mailto:sales@prodamus.ru)
* По телефону: 8 (495) 150-08-71

В обращении укажите, что вы планируете подключить платёжный виджет, и вам нужны merchantId и salesChannelId. После этого специалисты поддержки передадут вам необходимые данные.

### Пошаговая инструкция по подключению платёжного виджета

#### Шаг 1. Подключите скрипт платёжного виджета

Откройте HTML-код страницы сайта и перед закрывающим тегом \</head> добавьте следующий код:

```json
<script src="
https://widget.payform.ru/iframe.min.js
"></script>
```

Этот скрипт подключает функционал платёжного виджета Prodamus и делает его доступным на странице.

#### Шаг 2. Подготовьте код инициализации виджета

После подключения скрипта необходимо добавить на сайт код, в котором зафиксированы параметры заказа.

Пример базового кода:

```json
const prodamusWidget = new ProdamusWidget({
  merchantId: '{merchantId}',
  salesChannelId: '{salesChannelId}',
  currency: 'rub',
  products: [
    {
      name: 'Название товара',
      price: 12345,
      quantity: 1,
    },
  ],
});
```

{% hint style="info" %}
**Важно:**

Замените {merchantId} и {salesChannelId} на значения, полученные от технической поддержки Prodamus
{% endhint %}

#### Параметры платёжного виджета

При подключении платёжного виджета необходимо передать набор параметров заказа.

Часть параметров является обязательной — без них виджет не сможет работать. Остальные параметры дополнительные и используются при необходимости.

**Обязательные параметры платёжного виджета.** Эти параметры необходимо передавать всегда — без них платёжный виджет не откроется.

| **Параметр**         | **Что означает**                  | **Формат** | **Возможные значения**          |
| -------------------- | --------------------------------- | ---------- | ------------------------------- |
| merchantId           | Идентификатор мерчанта в Prodamus | String     | Выдаётся технической поддержкой |
| salesChannelId       | Идентификатор канала продаж       | String     | Выдаётся технической поддержкой |
| currency             | Валюта заказа                     | String     | rub, usd, eur, kzt              |
| products             | Список товаров в заказе           | Array      | Минимум один товар              |
| products\[].name     | Название товара                   | String     | Любая строка                    |
| products\[].price    | Цена товара                       | Number     | В единицах валюты               |
| products\[].quantity | Количество товара                 | Number     | Целое число                     |

**Дополнительные параметры платёжного виджета.** Эти параметры передаются при необходимости — для расширенной логики, учёта заказов или фискализации. Если вы не уверены, нужен ли параметр — его можно не указывать. Для стандартного приёма оплаты дополнительные параметры не требуются.

| **Параметр**            | **Что означает**                     | **Формат** | **Когда используется**                                          |
| ----------------------- | ------------------------------------ | ---------- | --------------------------------------------------------------- |
| merchantOrderNumber     | Номер заказа на стороне вашего сайта | String     | Для внутреннего учёта и сопоставления заказов                   |
| products\[].category    | Категория товара                     | String     | Сейчас используется значение «ОБУЧЕНИЕ», например для рассрочек |
| products\[].productType | Тип продукта                         | String     | На данный момент используется значение Purchase                 |
| products\[].taxType     | Ставка НДС                           | String     | Если требуется передавать данные для фискализации               |

Настройки полей ввода в платёжном виджете. С помощью параметра settings можно управлять тем, какие данные покупатель должен вводить при оплате, и какие значения можно подставить автоматически.

| **Параметр**             | **Что делает**                    | **Формат** | **Возможные значения** |
| ------------------------ | --------------------------------- | ---------- | ---------------------- |
| settings.hasEmailInput   | Требуется ли ввод e-mail          | Boolean    | true / false           |
| settings.emailInputValue | Предустановленное значение e-mail | String     | Любая строка           |
| settings.hasPhoneInput   | Требуется ли ввод телефона        | Boolean    | true / false           |
| settings.phoneInputValue | Предустановленный номер телефона  | String     | Любая строка           |

#### Шаг 3. Настройте открытие платёжного виджета

После добавления кода нужно настроить момент, когда будет открываться окно&#x20;

оплаты.Обычно платёжный виджет открывают при нажатии кнопки «Оплатить».

Для открытия виджета используйте команду:

prodamusWidget.open();

После этого:

* откроется платёжное окно Prodamus;
* покупатель сможет выбрать способ оплаты;
* платёж будет обработан стандартным образом.

### Итог

Платёжный виджет Prodamus позволяет принимать оплату прямо на сайте, без редиректов и сложных интеграций.

Для подключения необходимо:

1. Получить merchantId и salesChannelId в технической поддержке Prodamus.
2. Подключить скрипт виджета на сайт.
3. Добавить код с параметрами заказа.
4. Настроить открытие платёжного окна в нужный момент.

Такой способ интеграции подойдёт большинству сайтов и позволяет быстро запустить приём платежей.
