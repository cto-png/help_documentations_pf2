# Как создать секретный ключ для подключения интеграций

Для настройки интеграций Prodamus c внешними сервисами вам понадобится API-ключ. Чтобы получить его:

* Откройте канал продаж, который хотите интегрировать с внешним сервисом
* Перейдите в раздел «Интеграции»&#x20;
* Нажмите сгенерировать ключ

<figure><img src="../../.gitbook/assets/unknown (21).png" alt=""><figcaption></figcaption></figure>

Скопируйте и сохраните сгенерированный ключ. Его нужно вставить на стороне сервиса, с которым вы хотите настроить интеграцию.

{% hint style="info" %}
**Обратите внимание!** После закрытия модального окна просмотр ключа будет недоступен.&#x20;
{% endhint %}

<figure><img src="../../.gitbook/assets/unknown (22).png" alt=""><figcaption></figcaption></figure>

Если вы всё-таки потеряли доступ к секретному ключу, можно создать новый. Для этого нажмите кнопку «Перегенерировать».&#x20;

<figure><img src="../../.gitbook/assets/unknown (23).png" alt=""><figcaption></figcaption></figure>
