# amoCRM

Установите в «amoМаркете» — приложение Prodamus и создавайте платёжные ссылки из CRM-системы. Ниже — инструкция, как всё настроить.

### Шаг 1. Создайте секретный ключ в личном кабинета Prodamus

👉 [Инструкция: как авторизоваться на платёжной странице](https://help.prodamuspay.ru/)

Для настройки интеграций Prodamus c amoCRM вам понадобится API-ключ. Чтобы получить его:

* Откройте канал продаж, который хотите интегрировать с внешним сервисом
* Перейдите в раздел «Интеграции»&#x20;
* Нажмите сгенерировать ключ

<figure><img src="../../../.gitbook/assets/unknown (21).png" alt=""><figcaption></figcaption></figure>

Скопируйте и сохраните сгенерированный ключ.

{% hint style="info" %}
**Обратите внимание!** После закрытия модального окна просмотр ключа будет недоступен.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/unknown (22).png" alt=""><figcaption></figcaption></figure>

Созданный ключ нужно будет вставить в личном кабинете amoCRM.

### Шаг 2. Установите в amoCRM приложение Prodamus

Для этого авторизуйтесь в личном кабинете CRM-системы и откройте «amoМаркет».

![](<../../../.gitbook/assets/image (54)>)

Введите в поисковой строке «Prodamus».

![](<../../../.gitbook/assets/image (55)>)

Установите приложение «Prodamus / Продамус».

![](<../../../.gitbook/assets/image (56)>)

После установки приложение откроется автоматически.

### Шаг 3. Настройте приложение

**Настройки для вкладки «Магазины».**

![](<../../../.gitbook/assets/image (57)>)

1. Укажите в поле «Поддомен в Prodamus» ссылку на вашу платёжную страницу.

![](<../../../.gitbook/assets/image (58)>)

2. Вставьте секретный ключ, который вы ранее скопировали из личного кабинета Prodamus.

![](<../../../.gitbook/assets/image (59)>)

3. Выберите методы оплаты, которые будут доступны клиенту.

![](<../../../.gitbook/assets/image (60)>)

4. Выберите валюту, которой клиент сможет оплатить заказ.

![](<../../../.gitbook/assets/image (61)>)

5. Нажмите «Добавить магазин».

![](<../../../.gitbook/assets/image (62)>)

**Настройки для вкладки «Сделки».**

1. Откройте «Настройки полей».

![](<../../../.gitbook/assets/image (63)>)

2. Выберите в «Поле для ссылки» поле, которое будет использоваться в карточке сделки для платёжной ссылки Prodamus. Эта настройка не обязательна и используется только если вы хотите, чтобы в карточке сделки отображалась платёжная ссылка на заказ.

![](<../../../.gitbook/assets/image (64)>)

Если при нажатии на «Поле для ссылки» вы не можете ничего выбрать, значит, в карточке вашей сделки нет поля с типом «Ссылка» и его нужно создать.

👉 [Создание поля в amoCRM](https://www.google.com/url?q=https://www.amocrm.ru/support/field/create_a_field%23:~:text%3D%25D0%2592%25D0%25B0%25D0%25BC%2520%25D0%25BD%25D1%2583%25D0%25B6%25D0%25BD%25D0%25BE%2520%25D0%25B7%25D0%25B0%25D0%25B9%25D1%2582%25D0%25B8%2520%25D0%25B2%2520%25D1%2580%25D0%25B0%25D0%25B7%25D0%25B4%25D0%25B5%25D0%25BB,%25D0%25BD%25D0%25B5%25D1%2581%25D0%25BA%25D0%25BE%25D0%25BB%25D1%258C%25D0%25BA%25D0%25B8%25D1%2585%2520%25D0%25B2%25D0%25BE%25D1%2580%25D0%25BE%25D0%25BD%25D0%25BE%25D0%25BA%2520%25D0%25BD%25D0%25B0%2520%25D0%25BE%25D0%25BF%25D1%2580%25D0%25B5%25D0%25B4%25D0%25B5%25D0%25BB%25D0%25B5%25D0%25BD%25D0%25BD%25D1%258B%25D1%2585%2520%25D1%258D%25D1%2582%25D0%25B0%25D0%25BF%25D0%25B0%25D1%2585.\&sa=D\&source=editors\&ust=1685378325361696\&usg=AOvVaw2MmSyoOqSIyDmC7tdGXuh_)

3. Выберите в «Поле для суммы оплаты» поле, в которое будет подставляться сумма оплаченного заказа. По умолчанию это «Бюджет сделки».

![](<../../../.gitbook/assets/image (65)>)

4. Включите функцию «Работа с предоплатой» и выберите поле, в которое будет подставляться сумма предоплаты после её внесения. Эта настройка не обязательна и используется, только если вы берёте с клиентов предоплату.

![](<../../../.gitbook/assets/image (66)>)

Если при нажатии на «Поле для суммы сделки» вы не можете ничего выбрать, значит, в карточке вашей сделки нет поля с типом «Число» и его нужно создать.

👉 [Создание поля в amoCRMarrow-up-right](https://www.google.com/url?q=https://www.amocrm.ru/support/field/create_a_field%23:~:text%3D%25D0%2592%25D0%25B0%25D0%25BC%2520%25D0%25BD%25D1%2583%25D0%25B6%25D0%25BD%25D0%25BE%2520%25D0%25B7%25D0%25B0%25D0%25B9%25D1%2582%25D0%25B8%2520%25D0%25B2%2520%25D1%2580%25D0%25B0%25D0%25B7%25D0%25B4%25D0%25B5%25D0%25BB,%25D0%25BD%25D0%25B5%25D1%2581%25D0%25BA%25D0%25BE%25D0%25BB%25D1%258C%25D0%25BA%25D0%25B8%25D1%2585%2520%25D0%25B2%25D0%25BE%25D1%2580%25D0%25BE%25D0%25BD%25D0%25BE%25D0%25BA%2520%25D0%25BD%25D0%25B0%2520%25D0%25BE%25D0%25BF%25D1%2580%25D0%25B5%25D0%25B4%25D0%25B5%25D0%25BB%25D0%25B5%25D0%25BD%25D0%25BD%25D1%258B%25D1%2585%2520%25D1%258D%25D1%2582%25D0%25B0%25D0%25BF%25D0%25B0%25D1%2585.\&sa=D\&source=editors\&ust=1685378325363025\&usg=AOvVaw09o-JfBAt3TxfeDzFvBwFH)

5. Включите «Формировать ссылку на определённых этапах» и выберите этапы сделки, на которых можно будет формировать платёжные ссылки. Настройка не обязательна.

![](<../../../.gitbook/assets/image (67)>)

6. Нажмите на «Настройку воронок».

![](<../../../.gitbook/assets/image (68)>)

7. Выберите адрес вашей платёжной страницы в поле «Магазин (по-умолчанию)».

![](<../../../.gitbook/assets/image (69)>)

{% hint style="info" %}
Нюансы ☝️\
Если вы не можете указать адрес вашей платёжной страницы в поле «Магазин (по-умолчанию)», то:

* нажмите «Сохранить»;
* заново откройте приложение «Prodamus / Продамус»;
* повторите попытку указать адрес платёжной страницы.
{% endhint %}

8. Укажите, на какой этап будет перемещаться сделка после того, как клиент оплатит заказ.

![](<../../../.gitbook/assets/image (70)>)

### Шаг 4. Нажмите «Сохранить», чтобы закончить настройку приложения

![](<../../../.gitbook/assets/image (71)>)

### Шаг 5. Создайте платёжную ссылку

Для этого откройте карточку сделки и перейдите во вкладку «Prodamus».

![](<../../../.gitbook/assets/image (72)>)

Нажмите «Создать ссылку на оплату».

![](<../../../.gitbook/assets/image (73)>)

Укажите название продукта и его стоимость.

![](<../../../.gitbook/assets/image (74)>)

Выберите адрес платёжной страницы, через который будет происходить продажа, и нажмите «Создать ссылку».

![](<../../../.gitbook/assets/image (75)>)

Скопируйте платёжную ссылку и отправьте её клиенту.

![](<../../../.gitbook/assets/image (76)>)

Когда клиент внесёт платёж, статус ссылки автоматически поменяется на «Оплачено», а сделка переместиться на этап, который вы выбрали при настройке приложения Prodamus.
