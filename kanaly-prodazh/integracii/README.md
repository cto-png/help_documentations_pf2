# Интеграции со сторонними сервисами

Если вы хотите подключить интеграцию с Prodamus, это можно сделать несколькими способами — через готовые решения от наших партнёров или самостоятельно.

### Готовые интеграции

Готовые интеграции подготовлены нашими партнёрами и могут не поддерживать всех возможностей Prodamus. Если у вас возникли вопросы по работоспособности, функционалу и доработкам интеграций, рекомендуем адресовать их сервису, который подготовил интеграцию.

### Самостоятельная интеграция c Prodamus

Если вы не нашли нужный сервис в списке интеграций или ваша система написана вручную, вы можете настроить собственную интеграцию с Prodamus. Для этого прочитайте инструкции по ссылкам ниже.

👉 [Документация для самостоятельной интеграции ](https://help.prodamus.ru/payform/integracii/rest-api)

👉 [Техническая документация по автоплатежам](https://help.prodamus.ru/payform/integracii/tekhnicheskaya-dokumentaciya-po-avtoplatezham)
