# Toolsy

Вы можете подключить Prodamus к сервису Toolsy и предлагать клиентам платную подписку на ваш Telegram-канал. Ниже — инструкция, как всё настроить.

### Шаг 1. Запустите бота Toolsy <a href="#h.63ve71cl8hxh" id="h.63ve71cl8hxh"></a>

Для этого откройте Telegram, перейдите в [@toolsy\_bot](https://www.google.com/url?q=https://t.me/toolsy_bot\&sa=D\&source=editors\&ust=1684757398300200\&usg=AOvVaw3ua5bqIMEp4YcUplxALqkR) и нажмите «Запустить».

<figure><img src="../../../.gitbook/assets/image (61).png" alt=""><figcaption></figcaption></figure>

Выберите язык интерфейса.

<div align="left"><figure><img src="../../../.gitbook/assets/image (62).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Перейти к обучению».

<div align="left"><figure><img src="../../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure></div>

Посмотрите видеоинструкции, если хотите подробнее узнать, как работать с сервисом Toolsy. Чтобы продолжить настройку интеграции с Prodamus, нажмите «Завершить обучение».

<div align="left"><figure><img src="../../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Начать работу».

<div align="left"><figure><img src="../../../.gitbook/assets/image (65).png" alt=""><figcaption></figcaption></figure></div>

### Шаг 2. Создайте проект <a href="#h.ag1by551e565" id="h.ag1by551e565"></a>

Для этого выберите «Мои проекты» и нажмите «Создать проект».

<figure><img src="../../../.gitbook/assets/image (66).png" alt=""><figcaption></figcaption></figure>

Отправьте в чат название вашего проекта.

<div align="left"><figure><img src="../../../.gitbook/assets/image (67).png" alt=""><figcaption></figcaption></figure></div>

Проект создан. Теперь к нему нужно привязать вашего чат-бота. Если у вас уже есть чат-бот, отправьте его токен в чат с ботом Toolsy. Если у вас ещё нет бота, выполните описанные ниже действия.

### Шаг 3. Создайте чат-бота <a href="#h.pl6vox9gjhe5" id="h.pl6vox9gjhe5"></a>

Для этого откройте в Telegram [@botFather](https://www.google.com/url?q=https://t.me/BotFather\&sa=D\&source=editors\&ust=1684757398305075\&usg=AOvVaw1koKeAxT_4IoNG_OySUWIZ), нажмите «Запустить» и пропишите команду «/newbot».

<div align="left"><figure><img src="../../../.gitbook/assets/image (68).png" alt=""><figcaption></figcaption></figure></div>

Введите название чат-бота, которое будут видеть ваши клиенты.

<div align="left"><figure><img src="../../../.gitbook/assets/image (69).png" alt=""><figcaption></figcaption></figure></div>

Укажите username вашего бота, которое должно оканчиваться на bot или \_bot. Например, Prodamus\_bot.

<div align="left"><figure><img src="../../../.gitbook/assets/image (70).png" alt=""><figcaption></figcaption></figure></div>

Скопируйте токен созданного бота.

<div align="left"><figure><img src="../../../.gitbook/assets/image (71).png" alt=""><figcaption></figcaption></figure></div>

Отправьте токен в чат с ботом Toolsy.

<div align="left"><figure><img src="../../../.gitbook/assets/image (72).png" alt=""><figcaption></figcaption></figure></div>

Готово: бот подключён к проекту.

### Шаг 4. Подключите Toolsy к своему платному Telegram-каналу <a href="#h.m44dpgof6cod" id="h.m44dpgof6cod"></a>

Для этого перейдите в свой Telegram-канал, нажмите ⋮ и выберите «Управление каналом».

<figure><img src="../../../.gitbook/assets/image (73).png" alt=""><figcaption></figcaption></figure>

Откройте вкладку «Администраторы».

<div align="left"><figure><img src="../../../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Добавить администратора».

<div align="left"><figure><img src="../../../.gitbook/assets/image (75).png" alt=""><figcaption></figcaption></figure></div>

Сделайте администратором канала бота Toolsy.

<div align="left"><figure><img src="../../../.gitbook/assets/image (76).png" alt=""><figcaption></figcaption></figure></div>

Предоставьте боту права на добавление участников и администраторов. Нажмите «Сохранить».

<div align="left"><figure><img src="../../../.gitbook/assets/image (77).png" alt=""><figcaption></figcaption></figure></div>

Затем откройте чат с ботом Toolsy и нажмите на кнопку «Ресурсы».

<div align="left"><figure><img src="../../../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Добавить» → «Канал».

<div align="left"><figure><img src="../../../.gitbook/assets/image (79).png" alt=""><figcaption></figcaption></figure></div>

Затем перешлите любую публикацию из вашего канала в чат с ботом Toolsy.

<figure><img src="../../../.gitbook/assets/image9 (1).gif" alt=""><figcaption></figcaption></figure>

Готово: ваш Telegram-канал подключён к проекту в Toolsy.

<div align="left"><figure><img src="../../../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure></div>

### Шаг 5. Подключите свою платёжную страницу к проекту в Toolsy <a href="#h.n4zrsnn6dmp6" id="h.n4zrsnn6dmp6"></a>

Для настройки интеграций Prodamus c Toolsy вам понадобится API-ключ. Чтобы получить его:

* Откройте канал продаж, который хотите интегрировать с внешним сервисом
* Перейдите в раздел «Интеграции»&#x20;
* Нажмите сгенерировать ключ

<figure><img src="../../../.gitbook/assets/unknown (21).png" alt=""><figcaption></figcaption></figure>

Скопируйте и сохраните сгенерированный ключ.

{% hint style="info" %}
**Обратите внимание!** После закрытия модального окна просмотр ключа будет недоступен.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/unknown (22).png" alt=""><figcaption></figcaption></figure>

Вернитесь в чат с ботом Toolsy и кликните на «Платёжные системы».

<div align="left"><figure><img src="../../../.gitbook/assets/image (83).png" alt=""><figcaption></figcaption></figure></div>

Выберите «Prodamus (PayForm)».

<div align="left"><figure><img src="../../../.gitbook/assets/image (84).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Подключить».

<div align="left"><figure><img src="../../../.gitbook/assets/image (85).png" alt=""><figcaption></figcaption></figure></div>

Отправьте боту:

* Название вашей платёжной страницы. Если бы, например, URL-адрес вашей платёжной страницы был https://toolsy.payform.ru/, то вам нужно было взять из этой ссылки только toolsy — это и есть название.
* Через двоеточие без пробела после названия платёжной страницы — секретный ключ, который вы ранее скопировали из личного кабинета Prodamus.

{% hint style="info" %}
**Пример:** toolsy:2y2annke8018fniwuuq7tdkwmmuq7vwi4nzbr8z1182ftbn6p8mhw3bhz
{% endhint %}

<div align="left"><figure><img src="../../../.gitbook/assets/image (86).png" alt=""><figcaption></figcaption></figure></div>

Интеграция между Prodamus и Toolsy настроена. Осталось создать тариф и протестировать оплату.

<div align="left"><figure><img src="../../../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure></div>

### Шаг 6. Создайте тариф

Для этого выберите в чате с ботом «Продукты».

<figure><img src="../../../.gitbook/assets/1 копия.png" alt=""><figcaption></figcaption></figure>

Затем нажмите «Добавить продукт» или выберите ранее созданный продукт.

<figure><img src="../../../.gitbook/assets/2 копия (1).png" alt=""><figcaption></figcaption></figure>

Далее перейдите в «Тарифные планы».

<figure><img src="../../../.gitbook/assets/3 копия (1).png" alt=""><figcaption></figcaption></figure>

Нажмите «Добавить тариф».

<div align="left"><figure><img src="../../../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure></div>

Выберите:

* «Обычный тариф», если хотите, чтобы клиент сразу оплачивал подписку.
* «Пробный период», если хотите предоставить клиенту бесплатный доступ к подписке на определённый срок. Когда этот срок закончится, подписчику придёт уведомление с предложением платного продления подписки.

<div align="left"><figure><img src="../../../.gitbook/assets/image (90).png" alt=""><figcaption></figcaption></figure></div>

Отправьте боту название тарифа и выберите срок его действия.

<div align="left"><figure><img src="../../../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure></div>

Выберите валюту, в которой клиенты будут оплачивать тариф.

<div align="left"><figure><img src="../../../.gitbook/assets/image (92).png" alt=""><figcaption></figcaption></figure></div>

{% hint style="info" %}
**Важно!** Валюта оплаты, которую вы выбираете в Telegram-боте Toolsy, обязательно должна быть подключена к вашей платёжной странице в Prodamus, иначе интеграция не будет работать.
{% endhint %}

Отправьте боту стоимость тарифа, отметьте галочкой Telegram-канал, к которому покупатели будут получать доступ после оплаты, и нажмите «Готово».

<div align="left"><figure><img src="../../../.gitbook/assets/image (93).png" alt=""><figcaption></figcaption></figure></div>

Отметьте галочкой платёжные системы, через которые будет выполняться оплата тарифного плана. Для Prodamus это «Карта/PayForm». Нажмите «Готово».

<div align="left"><figure><img src="../../../.gitbook/assets/image (94).png" alt=""><figcaption></figcaption></figure></div>

### Шаг 7. Протестируйте оплату <a href="#h.57f72fwwk7k5" id="h.57f72fwwk7k5"></a>

Чтобы проверить, как работает оплата, откройте чат-бота, которого вы подключили к Toolsy, и нажмите «Запустить». Выберите язык интерфейса.

<div align="left"><figure><img src="../../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure></div>

Выберите ранее созданный тариф.

<div align="left"><figure><img src="../../../.gitbook/assets/image (96).png" alt=""><figcaption></figcaption></figure></div>

Нажмите кнопку «Оплатить».

<div align="left"><figure><img src="../../../.gitbook/assets/image (97).png" alt=""><figcaption></figcaption></figure></div>

Выберите способ оплаты «Карта/PayForm».

<div align="left"><figure><img src="../../../.gitbook/assets/image (98).png" alt=""><figcaption></figcaption></figure></div>

Нажмите «Перейти к оплате».

<div align="left"><figure><img src="../../../.gitbook/assets/image (99).png" alt=""><figcaption></figcaption></figure></div>

Прямо в Telegram откроется платёжная страница Prodamus, на которой нужно:

* указать свои контакты,
* нажать кнопку «Оплатить»,
* выбрать метод оплаты,
* внести платёж.

Когда платёж будет внесён, бот Toolsy оповестит вас об успешной оплате подписки.
